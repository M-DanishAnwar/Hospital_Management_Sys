# Hospital Management System

A comprehensive desktop application for managing hospital operations including patient registration, doctor management, appointments, medical records, and billing.

## Features

- **User Authentication**: Role-based access control (Admin, Doctor, Receptionist)
- **Patient Management**: Register, view, and manage patient records
- **Doctor Management**: Add, edit, and manage doctor profiles
- **Appointment Scheduling**: Book and manage patient appointments
- **Medical Records**: Store and retrieve patient medical history
- **Billing System**: Generate and manage patient bills
- **Admin Dashboard**: System overview with statistics and quick actions

## Technology Stack

- **Language**: Python 3.x
- **GUI Framework**: Tkinter
- **Database**: MySQL
- **Testing**: unittest (planned)
- **Version Control**: Git/GitHub
